Enable PostGIS (The "Spatial Brains")
By default, PostgreSQL only understands text and numbers. We need to "activate" the PostGIS extension so it understands coordinates and map geometries.
+1

Select your DB: In the left sidebar, click on your new v_neuron_db to highlight it.

Open the Query Tool:

Look at the top menu bar in pgAdmin. Click the icon that looks like a Magnifying Glass over a Table (or go to Tools > Query Tool).

Run the Activation Command:

In the empty white box that appears, type exactly this:

SQL
CREATE EXTENSION postgis;
Press the Execute/Refresh button (the "Play" button or F5).

You should see a message saying "Query returned successfully."

Verification:

Delete the previous text and type:

SQL
SELECT postgis_full_version();
Press Execute (F5) again.

In the "Data Output" tab below, you should see a long string starting with something like POSTGIS="3.4.2". If you see this, your database is now "map-aware".
