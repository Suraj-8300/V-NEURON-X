Since we are moving into Step 1.1: Base Road Network Extraction, we will use Python (OSMnx) to pull the data and then export it as a GeoPackage (.gpkg). This format is the "bridge" that allows you to open your exact road network in ArcGIS Pro for high-quality visualization and topological inspection.+1Step 1.1: Base Road Network MappingIn this step, we will download the "Drive" and "Walk" layers for the Nagpur Metropolitan Area (approx. $217\text{ km}^2$).1. The Python Extraction ScriptCreate a new Jupyter Notebook in your notebooks/ folder named 01_network_extraction.ipynb. Ensure your Python (V-NEURON) kernel is selected.Pythonimport osmnx as ox
import networkx as nx
import geopandas as gpd

# 1. Define the area of interest: Nagpur, Maharashtra
place_name = "Nagpur, Maharashtra, India"

# 2. Download the 'Drive' network (for cars/bikes)
print("Downloading drive network for Nagpur...")
drive_graph = ox.graph_from_place(place_name, network_type='drive', retain_all=False, simplify=True)

# 3. Download the 'Walk' network (for pedestrians/first-mile)
print("Downloading walk network for Nagpur...")
walk_graph = ox.graph_from_place(place_name, network_type='walk', retain_all=False, simplify=True)

# 4. Project the graphs to a UTM CRS (Universal Transverse Mercator) 
# This is essential for accurate distance/area calculations in ArcGIS
drive_graph_proj = ox.project_graph(drive_graph)
walk_graph_proj = ox.project_graph(walk_graph)

# 5. Save the raw graphs as .graphml files (for Python/NetworkX use later)
ox.save_graphml(drive_graph_proj, filepath="../data/nagpur_drive.graphml")
ox.save_graphml(walk_graph_proj, filepath="../data/nagpur_walk.graphml")

# 6. EXPORT FOR ARCGIS: Convert to GeoPackage
# This creates layers (nodes and edges) that ArcGIS can read natively
print("Exporting to GeoPackage for ArcGIS visualization...")
ox.save_graph_geopackage(drive_graph_proj, filepath="../data/nagpur_network.gpkg", encoding='utf-8')

print("Extraction Complete! Files saved in V-NEURON/data/")
2. ArcGIS Visualization WorkflowNow that you have the nagpur_network.gpkg file, follow these steps to visualize the V-NEURON foundation:Open ArcGIS Pro: Create a new Map project. Add Data: Drag and drop nagpur_network.gpkg from your data/ folder into the map. You will see two layers: edges (the road lines) and nodes (the intersections). Inspect Connectivity:Zoom into Sitabuldi Square or Zero Mile. Use the Symbolize tool to color the "edges" by their highway attribute (e.g., Primary roads in Red, Residential in Grey). Identify "Islands": Look for any road segments that aren't connected to the main network. In routing, these cause "Path not found" errors.
